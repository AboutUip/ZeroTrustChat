# 群组禁言技术规范

## 一、禁言流程

```
群主/管理员禁言成员:
1. 发送 GROUP_MUTE
2. 指定成员ID和禁言时长
3. MM1 记录禁言状态
4. 广播通知

被禁言成员:
- 无法发送消息
- 仍可接收消息
- 仍可查看群信息
```

## 二、禁言结构

```
┌─────────────────────────────────────────────────────────────┐
│  禁言记录 (MM1)                                               │
├─────────────────────────────────────────────────────────────┤
│  groupId: 群ID                                               │
│  userId: 被禁言用户ID                                         │
│  mutedBy: 禁言操作者ID                                        │
│  startTime: 禁言开始时间                                      │
│  duration: 禁言时长 (秒)                                      │
│  reason: 禁言原因 (可选)                                      │
└─────────────────────────────────────────────────────────────┘
```

## 三、禁言时长

| 时长 | 说明 |
|------|------|
| 60秒 | 1分钟 |
| 300秒 | 5分钟 |
| 1800秒 | 30分钟 |
| 3600秒 | 1小时 |
| 86400秒 | 24小时 |
| -1 | 永久禁言 |

## 四、解禁

```
解禁方式:
1. 手动解禁: 群主/管理员手动解除
2. 自动解禁: 禁言时长到期
3. 退群自动解除: 被禁言成员退群
```

## 五、权限检查

```
发送消息时:
1. 检查用户是否在群中
2. 检查用户是否被禁言
3. 禁言中 → 返回错误码 E_GROUP_MUTED
4. 未禁言 → 正常发送
```

## 六、权限级别

| 操作 | 群主 | 管理员 | 普通成员 |
|------|------|--------|----------|
| 禁言他人 | ✓ | ✓ | ✗ |
| 解禁他人 | ✓ | ✓ (仅自己禁的) | ✗ |
| 被禁言 | - | - | - |

## 七、通知

```
禁言时:
1. 通知被禁言成员
2. 通知群主/管理员
3. 群内其他成员不可见
```
