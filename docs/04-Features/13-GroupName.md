# 群组名称修改技术规范

## 一、功能

```
群主/管理员可修改群名称
```

## 二、消息流程

```
修改群名称:
1. 发送 GROUP_UPDATE (0x10)
2. 携带 TLV 扩展
3. MM1 验证权限
4. 更新 .zdb
5. 广播通知所有成员
```

## 三、TLV扩展

```
使用 ZSP TLV 扩展:

Type: 0x20 (GroupName)
Length: 可变
Value:
┌─────────────────────────────────────┐
│  GroupId: 群ID                      │
│  GroupName: 新群名 (UTF-8)          │
└─────────────────────────────────────┘
```

## 四、权限

```
可修改者:
- 群主
- 管理员

限制:
- 名称长度: 1-50 字符
- 禁止敏感词
- 修改频率: 1次/分钟
```

## 五、存储

```
存储位置: .zdb (加密)
结构:
- groupId
- groupName
- updateTime
- updateBy
```
