# 性能指标技术规范

## 一、系统容量

| 项目 | 指标 |
|------|------|
| 在线用户 | 100万 |
| 群组数量 | 每用户最多200个群 |
| 消息处理 | 100,000 QPS |
| 延迟 | P99 < 100ms |

## 二、资源消耗

| 资源 | 100万在线用户 |
|------|---------------|
| MM1 内存 | ~80 MB |
| MM2 消息缓存 | ~3 GB |
| .zdb 存储 | 按用户数 × 5MB |
| 网络带宽 | ~10 Gbps |

## 三、消息处理

| 指标 | 值 |
|------|-----|
| 单用户消息缓存 | 100条/会话 |
| 消息过期时间 | 7天 |
| 最大文件大小 | 100MB |
| 文件分片大小 | 64KB |

## 四、群组限制

| 指标 | 值 |
|------|-----|
| 单用户最大群数 | 200 |
| 单群最大成员 | 500 |
| 单群最大管理员 | 10 |
| 群消息密钥刷新 | 24小时 |

## 五、会话管理

| 指标 | 值 |
|------|-----|
| 会话超时 | 30分钟 |
| 心跳间隔 | 30秒 |
| 心跳超时 | 90秒 |
| 最大设备数 | 2/用户 |

## 六、安全参数

| 指标 | 值 |
|------|-----|
| 密钥刷新周期 | 24小时 |
| 认证限流 | 10次/分钟 |
| IP限流 | 5次/分钟 |
| 封禁时长 | 见 [02-Auth 封禁矩阵](../03-Business/02-Auth.md)（5 次起 15min～11+ 为 24h） |
| 滑动窗口 | 64 |

## 七、消息限制

| 指标 | 值 |
|------|-----|
| 消息编辑时限 | 5分钟 |
| 消息编辑次数 | 3次 |
| 好友请求过期 | 7天 |
| 群密钥确认超时 | 5分钟 |
