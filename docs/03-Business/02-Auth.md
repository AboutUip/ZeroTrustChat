# 认证安全机制技术规范

## 一、限流规则

```
┌─────────────────────────────────────────────────────────────┐
│  MM1 认证计数器                                               │
├─────────────────────────────────────────────────────────────┤
│  IP级别: 5次/分钟                                             │
│  用户级别: 10次/分钟                                           │
└─────────────────────────────────────────────────────────────┘
```

## 二、封禁机制

| 项目 | 值 |
|------|-----|
| 触发条件 | 连续5次认证失败 |
| 封禁时长 | 1小时 |
| 封禁范围 | userId + IP |
| 存储位置 | MM1内存 |

## 三、错误码

| 错误码 | 说明 |
|--------|------|
| E_AUTH_RATE_LIMIT | 频率超限 |
| E_AUTH_LOCKED | 已封禁 |
| E_AUTH_FAILED | 认证失败 |

## 四、认证成功

```
成功 → 重置MM1计数器 → 返回sessionId
```

## 五、安全原则

- 计数器/封禁存MM1内存
- 服务重启 → 状态丢失
- 不记录任何日志
- Level 2 销毁
